# TDD with PHPUnit

## Installation

* Install VirtualBox
* Install Vagrant
* Install Ansible
* Clone repository: `git clone git@github.com:sawyer-effect/tdd-php.git`

## To run VM

* Go to project dir: `cd tdd-php`
* Execute command `vagrant up`
